How to resolve file access error when restoring SQL Server DB

To resolve the “access denied” error when trying to restore SQL Server .BAK files:

Open services, and find the

SQL SERVER(MSSQLSERVER)

and change its “log on as” account from NT Service/MSSQLSERVER to Local System
