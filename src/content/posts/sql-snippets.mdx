### Getting identity column from last row added with a stored procedure

Stored procedure:

```sql
CREATE PROCEDURE [dbo].[qInsert]
    @Id int,
    @Name nvarchar(100)
AS
BEGIN
    SET NOCOUNT ON
    INSERT INTO [dbo].[artist]
    (
        [Name]
    )
    VALUES(
        @Name
    );
    RETURN SCOPE_IDENTITY();
END
GO
```

```sql
EXEC @rp = qInsert 0, 'Bobby Fuller Four'
SELECT @rp as 'identify'
```
